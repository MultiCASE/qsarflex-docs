# QSAR Flex Docs

# What is QSAR Flex?

 is a software platform that provides high quality (Q)SAR models, databases, read across modules and various tools for the application of novel computational approaches for risk assessment of chemical compounds and is especially suitable for complex toxicological end points.

Now all this goodness in Web!

**Key Features:**

- nitrosamine Drug Substance Related Impurities (NDSRIs).
- : Uses structural searches and descriptor calculations for evidence-based toxicity predictions.
- : Offers solutions for pharmaceutical, chemical, and other fields needing chemical safety assessment.
