# Model Catalog

The QSAR Flex model catalog lists all available prediction endpoints and their availability across the Web and Desktop platforms. Contact [support@multicase.com](mailto:support@multicase.com) to add modules to your license.

---

## N-Nitrosamine & NDSRI Evaluation

| Module | Platform | Description |
|---|---|---|
| **CPCA Prediction** | Web + Desktop | Assigns N-nitrosamine impurities (including NDSRIs) to a predicted carcinogenic potency category with acceptable intake (AI) limits. Includes surrogate search for compounds with animal carcinogenicity data and regulatory AI precedents. |
| **Surrogate Search** | Desktop only | Legacy module for finding N-nitrosamine surrogates with available animal carcinogenicity data. Preceded CPCA; retained for backward compatibility. |
| **Cross Similarity** | Desktop only | Generates an NxN structural similarity matrix across your entire compound library using fingerprint-based comparison. |

## Genotoxicity

| Module | Platform | Description |
|---|---|---|
| **Ames Mutagenicity** | Web + Desktop | Predicts bacterial reverse mutation (Ames test) outcome using structure-activity relationship models. |

## Ecotoxicity

| Module | Platform | Description |
|---|---|---|
| **Tetrahymena 48h GC50** | Web + Desktop | Predicts 48-hour growth concentration causing 50% inhibition in *Tetrahymena pyriformis* — a standard aquatic toxicity endpoint. |
| **Soil Adsorption** | Web + Desktop | Predicts the organic carbon-normalized soil adsorption coefficient (Koc) for environmental fate assessment. |

---

## Physicochemical Properties

| Module | Platform | Description |
|---|---|---|
| **Water Solubility** | Web + Desktop | Predicts aqueous solubility (log S) at physiological pH. |
| **LogP** | Web + Desktop | Predicts the octanol-water partition coefficient, a key indicator of lipophilicity and membrane permeability. |
