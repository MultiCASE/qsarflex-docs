## **Licensing Options**

We offer flexible licensing models to suit various needs:

**Subscription-Based Licensing**: Access the platform with an unlimited number of tests within a specified period, offering the flexibility to scale usage as needed.

**Pay-Per-Test Licensing**: For a usage-based approach, purchase tests in groups of 20 or more, providing a cost-effective solution for occasional or limited use.

**Enterprise Solutions**: Tailored enterprise licensing accommodates multiple users and extensive testing needs, ensuring comprehensive support for larger organizations.

**Module-Specific Licensing:** Licenses can be customized to include only the specific modules you require, allowing you to purchase access to selected functionalities without the need to acquire all available modules.

**Trial Licensing:** A limited-time, no-cost option to explore the interface and get hands-on experience using the tool to evaluate complex toxicological endpoints. Get an idea to see if the QSAR Flex Web App aligns with your _in silico_ needs.
